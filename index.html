Hypixel AutoWoot

PlugDJ script

Download this project as a .zip file Download this project as a tar.gz file

General Information:

The Hypixel AutoWoot is a script used to automatically woot songs that are being played. It's easy to use and easy to set up.

To toggle the AutoWoot script just double click on the woot button

Installation:

Installation is very easy, simply copy the code below and save it as a bookmark for your browser.

javascript: (function (){var jsCode = document.createElement('script'); jsCode.setAttribute('id', 'AutoWoot'); jsCode.setAttribute('src', 'https://raw.github.com/xmannj2/AutoWoot/master/HypixelWoot.js'); document.body.appendChild(jsCode); }());

When you're done saving it as a bookmark then click on the bookmark while on Plug.dj screen. It should give you a greeting if it worked.
